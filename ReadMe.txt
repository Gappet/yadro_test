Данное решение собирается при помощи программы cmake.
Компилирование должно осуществляться mingw.

Команды для сборки:
cmake -G "MinGW Makefiles" ../Sheet
cmake --build .

Программа читает произвольную CSV-форму из файла, вычисляет значения ячеек, если это необходимо, и 
выводит получившуюся табличку в виде CSV-представления в консоль.

После сборки программы программа запускается следующим образом:
	Sheet.exe name_of_table.csv

, где name_of_table.csv имя файла, файл должен находиться в одной папке с Sheet.exe.

Программа обрабатывает каждую ячейку таблицы и если ячейка содержит ошибку, выводит код ошибки.
Коды ошибок ячеек:
	InV - невалидное значение ячейки(вместо числа содержит строку , пустая ячейка);
	InF - невалидная формула (формула содержит невалидный адрес ячейки);
	CR - формула содержит циклическую зависимость;
	DIV0 - деление на ноль.

Если файл пустой, то программа выведет: Empty table

Если исходная таблица имеет неверные номера строк, то программа выведет: Invalid row value

Если таблица имеет некорректный формат: Invalid format table.


В папке example содержаться несколько примеров:
add_cell.csv - таблица из технического задания;
circle_table.csv - таблица с формулами которые содержат циклические зависимости;
empty_table.csv - пустая таблица;
invalid_formula.csv - таблица содержит невалидные ячейки;
simple_table.csv - простая таблица;
test_all_math.csv - таблица содержит все допустимые математические действия;
test_all_math2.csv - таблица содержит все допустимые математические действия;
gibrid_formula.csv - таблица содержит формулы с числовыми константами;
invalid_row_value.csv - нумерация строк в таблице неверна.
invalid_format_table.csv - таблица имеет некорректный формат
